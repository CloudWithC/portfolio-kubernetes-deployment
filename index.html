CI/CD Docker Deployment on AWS

          

            Automated the full deployment lifecycle — Docker image build, ECR push, and EC2 deploy —

Infrastructure Automation using Terraform on AWS

          

            Provisioned 5 AWS resources (EC2, Security Groups, IAM Role, IAM Policy, VPC) through a single

            peer-reviewable infra changes, and executed the full Terraform lifecycle (init → plan →
